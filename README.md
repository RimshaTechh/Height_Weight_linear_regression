Height Prediction Using Simple Linear Regression

Project Overview

This project uses Simple Linear Regression to predict a person's height based on weight.

Dataset

The dataset contains two columns:

- Weight
- Height

Libraries Used

- NumPy
- Pandas
- Matplotlib
- Scikit-learn

Steps Performed

1. Loaded the dataset.
2. Visualized the data using a scatter plot.
3. Trained a Linear Regression model.
4. Predicted height for a given weight.
5. Evaluated the model using R² score.
6. Plotted the regression line.

Conclusion

The model learned the relationship between weight and height and can predict height based on weight using Simple Linear Regression.
